记录一次tf缺少cuDNN dll的问题
使用tensorflow的时候遇到了这个问题：Loaded cuDNN version 8302 Could not load library cudnn_cnn_infer64_8.dll. Error code 126 但实际上我已经把cudnn_cnn_infer64_8.dll加到环境变量中去了。 这可能是由于tf和cuDNN版本不匹配造成的问题。 可以考虑强行引入动态链接库： 12import ctypesctypes.WinDLL('path/to/cudnn_cnn_infer64_8.dll') 这样tf就可以正常训练啦！
